# 🚗 Người 1 — Vehicle System

**Module:** Vehicle System
**Công việc:** Vehicle hierarchy, movement

## Cấu trúc thư mục

```
src/com/smartcity/traffic/vehicle/   ← Toàn bộ code Java của bạn
docs/Vehicle_System_Report.md        ← Báo cáo của bạn
```

## Hướng dẫn

1. Clone repo về máy
2. Checkout nhánh này: `git checkout feature/vehicle-system`
3. Đặt code Java vào `src/com/smartcity/traffic/vehicle/`
4. Đặt báo cáo vào `docs/Vehicle_System_Report.md`
5. Commit và push lên nhánh này

## Các lớp cần làm

- `Vehicle.java` — Lớp cha trừu tượng
- `Car.java`, `Motorbike.java`, `Truck.java`, `Bus.java`, `Ambulance.java`
- Hỗ trợ: hierarchy (kế thừa), movement (di chuyển)
