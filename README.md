# 🎨 Người 5 — GUI & Multimedia

**Module:** GUI & Multimedia
**Công việc:** JavaFX, sprite, sound, animation

## Cấu trúc thư mục

```
src/com/smartcity/gui/               ← Toàn bộ code Java của bạn
assets/sprites/                      ← Ảnh sprite phương tiện
assets/sounds/                       ← File âm thanh
docs/GUI_Multimedia_Report.md        ← Báo cáo của bạn
```

## Hướng dẫn

1. Clone repo về máy
2. Checkout nhánh này: `git checkout feature/gui-multimedia`
3. Đặt code Java vào `src/com/smartcity/gui/`
4. Đặt assets vào thư mục `assets/`
5. Đặt báo cáo vào `docs/GUI_Multimedia_Report.md`
6. Commit và push lên nhánh này

## Các lớp cần làm

- `MainWindow.java` — Cửa sổ chính JavaFX
- `MapRenderer.java` — Vẽ bản đồ
- `VehicleSprite.java` — Hiển thị phương tiện
- `SoundManager.java` — Quản lý âm thanh
- `AnimationEngine.java` — Vòng lặp animation
